Politique de Confidentialité

Cette politique explique comment notre projet d'automatisation interne collecte, utilise et protège vos données en relation avec les API Google.

1. Utilisation des données Google (Google API Services User Data Policy)

Notre application utilise les API Google de manière restreinte pour accomplir sa tâche d'automatisation via la plateforme Make. L'utilisation des informations reçues par les API Google et leur transfert vers toute autre application se conforment aux Règles d'utilisation des données des services d'API Google, y compris les exigences d'utilisation restreinte (Limited Use requirements).

2. Quelles données collectons-nous et pourquoi ?

  • Gmail (Lecture) : L'application accède à la boîte de réception uniquement pour identifier les e-mails pertinents contenant les pièces jointes à traiter.
  • Google Drive (Écriture) : L'application utilise l'accès à Drive uniquement pour déposer (téléverser) les pièces jointes extraites dans un dossier spécifique.

3. Comment les données sont-elles stockées et partagées ?

Aucune donnée (ni le contenu des e-mails, ni les pièces jointes, ni les identifiants) n'est stockée sur ce site web ou sur une base de données tierce autre que le traitement temporaire effectué par la plateforme Make au moment de l'exécution du scénario. Aucune donnée personnelle n'est revendue, partagée à des annonceurs ou exploitée à des fins commerciales.

4. Révocation des accès

Vous pouvez à tout moment révoquer l'accès de cette application à votre compte Google en vous rendant sur la page des paramètres de sécurité de votre compte Google (gérer les applications tierces).
